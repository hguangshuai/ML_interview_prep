## 📋 问题内容总结

### 核心概念
**偏差-方差权衡** = "准确度 vs 稳定性"

#### 中文理解
- **偏差**：系统性误差，模型过于简单
- **方差**：随机性误差，模型过于复杂
- **权衡**：需要在准确性和稳定性之间找到平衡

#### 英文标准答案
- **Bias**: Systematic error from oversimplified assumptions
- **Variance**: Error from sensitivity to training data variations
- **Tradeoff**: Balance between accuracy and stability

### 数学表示
```
总误差 = 偏差² + 方差 + 噪声
```

### 四种状态
1. **低偏差 + 低方差**：理想状态
2. **低偏差 + 高方差**：过拟合
3. **高偏差 + 低方差**：欠拟合
4. **高偏差 + 高方差**：最差状态
